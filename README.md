### 1. 数组

- 对于有序或局部有序的一维数组，要想到二分查找算法；

- 遍历二维数组，用行指针和列指针，通过行列指针移动定位某个特定的元素；

- 对于元素特殊的数组，注意寻找特殊点，比如有序的临界点，二维数组的四个角；
